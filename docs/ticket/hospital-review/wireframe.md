# 病院向けレビュー機能 - ワイヤーフレーム

## 画面一覧

1. 名前入力画面（初回アクセス時）
2. 制作物一覧画面
3. レビュー画面（ステップ一覧 + フィードバック入力）

---

### 要素詳細

| 要素           | 説明                   |
| -------------- | ---------------------- |
| 病院アイコン   | シンプルな病院アイコン |
| 病院名         | DBから取得して表示     |
| 名前入力欄     | テキスト入力、必須     |
| 開始するボタン | 入力が空の場合は非活性 |

### 状態

| 状態     | 動作                           |
| -------- | ------------------------------ |
| 入力なし | ボタン非活性（グレー）         |
| 入力あり | ボタン活性（プライマリカラー） |
| 送信中   | ボタンにローディング表示       |

### 要素詳細

| 要素             | 説明                           |
| ---------------- | ------------------------------ |
| 戻るボタン       | 制作物一覧に戻る               |
| 制作物タイトル   | 中央に表示                     |
| Step番号         | 各ステップのヘッダー           |
| ボード画像       | クリックで拡大表示（モーダル） |
| テキスト         | ナレーション/テロップの内容    |
| フィードバック欄 | テキストエリア、3行程度        |
| 保存ボタン       | 固定フッター or スクロール追従 |
| 保存済み表示     | 最終保存時刻                   |

### インタラクション

| 操作               | 動作                                 |
| ------------------ | ------------------------------------ |
| 画像クリック       | モーダルで拡大表示                   |
| フィードバック入力 | リアルタイムで入力可能               |
| 保存ボタン         | 全ステップのフィードバックを一括保存 |
| ページ離脱時       | 未保存の場合は確認ダイアログ         |

## 6. カラー・スタイルガイド（参考）

### カラーパレット

| 用途         | カラー                | 備考               |
| ------------ | --------------------- | ------------------ |
| プライマリ   | `#3B82F6` (blue-500)  | ボタン、アクセント |
| 成功         | `#22C55E` (green-500) | 送信済バッジ       |
| 警告         | `#F59E0B` (amber-500) | 未保存警告         |
| テキスト     | `#1F2937` (gray-800)  | 本文               |
| サブテキスト | `#6B7280` (gray-500)  | 補足情報           |
| 背景         | `#F9FAFB` (gray-50)   | ページ背景         |
| カード背景   | `#FFFFFF`             | カード             |
| ボーダー     | `#E5E7EB` (gray-200)  | 区切り線           |

### コンポーネントスタイル

| コンポーネント       | スタイル                                                      |
| -------------------- | ------------------------------------------------------------- |
| カード               | `shadow-sm`, `rounded-lg`, `border`                           |
| ボタン（プライマリ） | `bg-blue-500`, `text-white`, `rounded-md`, `px-4 py-2`        |
| テキストエリア       | `border`, `rounded-md`, `focus:ring-2`, `focus:ring-blue-500` |
| バッジ（レビュー待） | `bg-gray-100`, `text-gray-600`, `text-sm`                     |
| バッジ（レビュー済） | `bg-green-100`, `text-green-700`, `text-sm`                   |
